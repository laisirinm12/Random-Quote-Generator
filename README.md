https://random-quote-generator-1mle.onrender.com/quote
---

This is a simple **Random Quote Generator** built using **HTML, CSS, JavaScript and a Node.js API**.  
The application displays random quotes along with the author's name and dynamically changes the **theme and background** based on the quote category.

Users can generate new quotes, copy them to the clipboard, or download the quote as an image.

## Features

- Random quote generation using a custom API
- Dynamic background themes based on quote category
- Copy quote to clipboard
- Download quote as an image
- Glassmorphism styled quote card
- Smooth animations and responsive layout

## Categories

Quotes are grouped into themes that automatically change the background:

- **Motivation**
- **Love**
- **Wisdom**
- **Coding**

Each category applies a different background style to enhance the experience.

## Technologies Used

**Frontend**
- HTML5
- CSS3
- JavaScript

**Backend**
- Node.js
- Express.js

**Libraries**
- html2canvas (for downloading quotes as images)
- CORS

## Purpose

This project was created to practice:

- Building a simple **REST API**
- Connecting **frontend with backend**
- Dynamic UI updates using JavaScript
- Implementing interactive features like **copy and download**
- Structuring a small **full-stack web application**
